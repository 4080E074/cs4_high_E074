機密性(Confidentiality)

使資訊不可用或不揭露給未經授權之個人、個體或者團體，保護資訊不被非法存取或揭露，先要做好資訊分級標示，分類後要標示等級。
```
```
完整性(Integrity)

保護資產的準確度(accuracy)和完全性(completeness)的性質，確保資產能正常運作及資訊在任何階段沒有不適當的修改或損毀。
(如網頁被駭，就是完整性的問題)
```
```
可用性(Availability )

經授權個體因應需求之可存取即可使用的及經授權的使用者能適時的存取所需資訊性質。(如硬體故障，使得授權的使用者無法使用，
就是可用性的問題)
